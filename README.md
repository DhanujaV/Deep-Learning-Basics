# Handwritten Digit Classification Using Perceptron (MNIST Dataset)

## Overview

This project demonstrates the implementation of a Perceptron model for handwritten digit recognition using the MNIST dataset. The model learns to classify grayscale images of handwritten digits (0–9) and serves as an introduction to neural networks and deep learning concepts.

The project covers data preprocessing, model training, prediction, and performance evaluation using the MNIST dataset.

---

## Dataset

### MNIST Dataset

The MNIST (Modified National Institute of Standards and Technology) dataset is one of the most widely used datasets for machine learning and deep learning.

#### Dataset Details

- 70,000 handwritten digit images
- 60,000 training images
- 10,000 testing images
- 10 classes (digits 0–9)
- Image size: 28 × 28 pixels
- Grayscale images

## Project Objectives

- Understand the fundamentals of neural networks.
- Implement a Perceptron-based classifier.
- Train a model using the MNIST dataset.
- Perform handwritten digit recognition.
- Evaluate model performance on unseen data.
